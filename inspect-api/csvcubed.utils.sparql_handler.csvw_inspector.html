csvcubed.utils.sparql_handler.csvw_inspector¶

Code List Inspector¶

Provides access to inspect the contents of an rdflib graph containing one of more code lists.

class csvcubed.utils.sparql_handler.csvw_inspector.CsvWInspector(rdf_graph: ConjunctiveGraph, csvw_json_path: Path)¶

Bases: object

Provides access to inspect the contents of an rdflib graph containing one of more code lists.

property catalog_metadata: List[CatalogMetadataResult]¶

Returns a list of catalog metadata results such as title, label, issue/modification date and time etc. This supports each result also having the graph_uri that the dcat:Dataset was defined in with it.

property column_definitions: Dict[str, List[ColumnDefinition]]¶

Map of csv_url to the list of column definitions for the given CSV file.

csvw_json_path: Path¶
property csvw_type: CSVWType¶

Returns the type of the primary graph in a csvw. E.g. CSVWType.CodeList or CSVWType.QbDataSet

get_column_definitions_for_csv(csv_url: str) → List[ColumnDefinition]¶

Returns the `ColumnDefinition`s for a given csv file, raises a KeyError if the csv_url is not associated with a ColumnDefinition.

get_primary_catalog_metadata() → CatalogMetadataResult¶

Retrieves the catalog metadata that is specifically only defined in the primary graph.

get_table_info_for_csv_url(csv_url: str) → TableSchemaPropertiesResult¶

Retrieves the stored result of the table schema properties cached property.

primary_graph_uri: str¶
rdf_graph: ConjunctiveGraph¶
