csvcubed.cli.error_mapping module¶

csvcubed.cli.error_mapping.friendly_error_mapping(error: ValidationError) → str¶

Given a validation error it returns an error message that is tailored to the qube-config.json interface so it’s more user-friendly.

Note: adding any new error will require a test in ‘test_build_friendly_error_msgs.py’
