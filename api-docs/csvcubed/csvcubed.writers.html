URIs declared in the columns section of the CSV-W are relative to the CSV’s location. URIs declared in the JSON-LD metadata section of the CSV-W are relative to the metadata file’s location.

This function makes both point to the same base location - the CSV file’s location. This ensures that we can talk about the same resources in the columns section and the JSON-LD metadata section.
