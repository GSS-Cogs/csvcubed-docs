Pandas¶

Functions to help when working with pandas dtypes.

csvcubed.utils.pandas.coalesce_on_uri_safe(data: Optional[Union[pandas.core.frame.DataFrame, pandas.core.series.Series]]) → Optional[Union[pandas.core.frame.DataFrame, pandas.core.series.Series]][source]¶

Coalesce a categorical Pandas Series() or DataFrame so that all uri_safe collisions have the same value. The value which is preferred is the first value when the list of colliding category names are sorted() i.e. ‘canada’ and ‘Canada’ categories will replace ‘canada’ with ‘Canada’
